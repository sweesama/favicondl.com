Descripción general

Mzu favicondl proporciona plantillas de URL de favicon listas para copiar y una interfaz de extracción de favicons. Por defecto, utiliza un Cloudflare Worker para descubrir iconos reales y proxy de imágenes para descargas.

Obtención optimizada para vista previa rápida

Gratis

Sin registro necesario

Confiable

Diseñado para alta disponibilidad

Autenticación

Este proyecto utiliza plantillas de URL de servicios de favicon gratuitos, no se requiere autenticación ni API Key.

Nota: Aunque no requerimos API Key, por favor usa el servicio de manera razonable para evitar solicitudes excesivas.

Endpoints

GET Google Favicon URL

Obtener el favicon de un dominio específico

URL base:

https://www.google.com/s2/favicons

Ejemplo completo:

https://www.google.com/s2/favicons?domain=example.com&sz=128

Parámetros

Nombre del parámetro Tipo Requerido Valor predeterminado Descripción
domain string Sí - El dominio del cual obtener el favicon, ej. example.com
sz number No 32 El tamaño del favicon, ej. 16, 32, 64, 128

Detalles de parámetros:

dominio

Soporta múltiples formatos de dominio:

  • • example.com
  • • www.example.com
  • • https://example.com
  • • sub.example.com
tamaño

Opciones de tamaño de favicon:

  • • 16 - 16x16
  • • 32 - 32x32
  • • 64 - 64x64
  • • 128 - 128x128

Respuestas

Respuesta exitosa (formato JSON)

Respuesta JSON 
{
  "success": true,
  "domain": "example.com",
  "url": "https://www.google.com/s2/favicons?domain=example.com&sz=32",
  "size": "small",
  "format": "ico",
  "width": 32,
  "height": 32,
  "timestamp": "2026-01-15T10:30:00Z"
}

Respuesta de error

Respuesta de error 
{
  "success": false,
  "error": "Domain not found",
  "message": "Unable to find favicon for the specified domain",
  "code": 404,
  "timestamp": "2026-01-15T10:30:00Z"
}

Nota: Cuando se usa format=redirect, la API redirige directamente a la URL del favicon sin devolver una respuesta JSON.

Ejemplos

JavaScript (Fetch)

JavaScript 
function getFaviconUrl(domain, sz = 32) {
  return `https://www.google.com/s2/favicons?domain=${domain}&sz=${sz}`;
}

const url = getFaviconUrl('example.com', 128);
console.log(url);

const img = document.createElement('img');
img.src = url;
img.width = 32;
img.height = 32;
document.body.appendChild(img);

Python (Requests)

Python 
def get_favicon_url(domain: str, sz: int = 32) -> str:
    return f"https://www.google.com/s2/favicons?domain={domain}&sz={sz}"

print(url)

Errores

La API devuelve el código de estado HTTP y mensaje de error correspondientes cuando ocurre un error.

Código de estado HTTP Código de error Descripción Solución
400 invalid_domain Formato de dominio inválido Verifica el formato del dominio y asegúrate de que sea válido
404 domain_not_found Dominio no encontrado Confirma que el dominio es correcto y el sitio web es accesible
404 favicon_not_found Favicon no encontrado El sitio web puede no tener un favicon configurado
429 rate_limit_exceeded Límite de solicitudes excedido Reduce la frecuencia de solicitudes e intenta más tarde
500 internal_error Error interno del servidor El servicio no está disponible temporalmente, intenta más tarde
