개요

Mzu favicondl은 복사 가능한 파비콘 URL 템플릿과 파비콘 추출 UI를 제공합니다. 기본적으로 Cloudflare Worker를 사용하여 실제 아이콘을 찾고 다운로드를 위해 이미지를 프록시합니다.

인증

이 프로젝트는 무료 파비콘 서비스 URL 템플릿을 사용하며, 인증이나 API 키가 필요하지 않습니다.

참고: API 키는 필요하지 않지만, 과도한 요청을 피하기 위해 합리적으로 사용해 주세요.

엔드포인트

GET Google Favicon URL

지정된 도메인의 파비콘 가져오기

기본 URL:

https://www.google.com/s2/favicons

전체 예시:

https://www.google.com/s2/favicons?domain=example.com&sz=128

파라미터

파라미터 이름 유형 필수 기본값 설명
domain string 예 - 파비콘을 가져올 도메인 (예: example.com)
sz number 아니오 32 파비콘 크기 (예: 16, 32, 64, 128)

파라미터 상세:

도메인

여러 도메인 형식 지원:

  • • example.com
  • • www.example.com
  • • https://example.com
  • • sub.example.com
크기

파비콘 크기 옵션:

  • • 16 - 16x16
  • • 32 - 32x32
  • • 64 - 64x64
  • • 128 - 128x128

응답

성공 응답 (JSON 형식)

JSON 응답 
{
  "success": true,
  "domain": "example.com",
  "url": "https://www.google.com/s2/favicons?domain=example.com&sz=32",
  "size": "small",
  "format": "ico",
  "width": 32,
  "height": 32,
  "timestamp": "2026-01-15T10:30:00Z"
}

오류 응답

오류 응답 
{
  "success": false,
  "error": "Domain not found",
  "message": "Unable to find favicon for the specified domain",
  "code": 404,
  "timestamp": "2026-01-15T10:30:00Z"
}

참고: format=redirect 사용 시 API는 JSON 응답을 반환하지 않고 파비콘 URL로 직접 리다이렉트합니다.

사용 예시

JavaScript (Fetch)

JavaScript 
function getFaviconUrl(domain, sz = 32) {
  return `https://www.google.com/s2/favicons?domain=${domain}&sz=${sz}`;
}

const url = getFaviconUrl('example.com', 128);
console.log(url);

const img = document.createElement('img');
img.src = url;
img.width = 32;
img.height = 32;
document.body.appendChild(img);

Python (Requests)

Python 
def get_favicon_url(domain: str, sz: int = 32) -> str:
    return f"https://www.google.com/s2/favicons?domain={domain}&sz={sz}"

url = get_favicon_url('example.com', 128)
print(url)

오류

오류 발생 시 API는 해당 HTTP 상태 코드와 오류 메시지를 반환합니다.

HTTP 상태 코드 오류 코드 설명 해결 방법
400 invalid_domain 잘못된 도메인 형식 도메인 형식을 확인하고, 유효한 도메인인지 확인하세요
404 domain_not_found 도메인을 찾을 수 없음 도메인이 올바르고 웹사이트에 접근 가능한지 확인하세요
404 favicon_not_found 파비콘을 찾을 수 없음 웹사이트가 파비콘을 설정하지 않았을 수 있습니다
429 rate_limit_exceeded 요청 제한 초과 요청 빈도를 줄이고 나중에 다시 시도하세요
500 internal_error 내부 서버 오류 서비스가 일시적으로 이용할 수 없습니다. 나중에 다시 시도하세요
